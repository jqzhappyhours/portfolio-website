# Portfolio Website

Personal portfolio site showcasing my projects, starting with [Pole Shape Classifier](https://github.com/jqzhappyhours/pole-shape-classifier).

Plain HTML/CSS/JS — no build step, no dependencies.

## Run locally

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000

## Deploy

Static site, deployable as-is to GitHub Pages, Vercel, or Netlify.
